netpulse

IP Geolocation Aggregation API

что это

netpulse агрегирует данные геолокации IP-адресов из 7 бесплатных источников с параллельным запросом и консенсусным голосованием. результат — самая точная оценка на основе медианы координат и моды страны.

работает  |  swagger  |  github

эндпоинты

  • GET /api/v1/ip/{ip} геолокация IP
  • GET /api/v1/asn/{asn} информация об ASN
  • GET /api/v1/subnet/{cidr} информация о подсети
  • POST /api/v1/batch пакетный запрос
  • GET /health состояние источников
  • GET /metrics метрики сервера

быстрый старт

запрос:

curl https://netpulse.d3vo.ru/api/v1/ip/8.8.8.8
